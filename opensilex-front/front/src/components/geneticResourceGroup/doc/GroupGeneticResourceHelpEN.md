# GeneticResource Groups help

Author: Maximilian Hart - Opensilex (16/08/2023)

## Create geneticResource groups

A group of genetic resource is a collection of different genetic resource.
It has a name and can have a description.
Its purpose is to improve the ergonomics of the interface and the user experience.

## Its use

The group of genetic resource allows us to classify genetic resource by categories of our choice.

This concept is therefore used for filtering genetic material, but also scientific objects. For example, until now to
filter out scientific objects whose genetic resources are not descendants of the Syrah type vine, we would have had to select each
genetic resource individually. With this new concept, this search is much easyer because we could just create a Groupe-Syrah that
would contain all the Syrah type genetic resources.

Au niveau de module 'GeneticResource', nous pouvons chercher un groupe et visualiser les ressources génétiques qu'il contient, ou
inversement, chercher les ressources génétiques et visualiser à quel groupe appartient-il.

On the 'GeneticResource' tab, we can search by group to see what genetic resources are contained in that group, or vise-versa, search a genetic resource to see what groups it is part of.
