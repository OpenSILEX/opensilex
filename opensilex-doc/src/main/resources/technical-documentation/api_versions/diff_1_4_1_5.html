Changelog

Versions⇑

Changes from 1.4.10 to 1.5.

What's New⇑

  1. GET/core/agroportal/ping Ping agroportal server
  2. GET/core/agroportal/search Search through agroportal
  3. GET/core/agroportal/ontologies Get ontologies from agroportal
  4. GET/core/data/batch_history/{uri} Get batch
  5. DELETE/core/data/batch_history/{uri} Delete batch history by URI
  6. GET/core/data/batch_history Search data batch history
  7. POST/core/datafiles/upload-dx Upload and parse DX file
  8. POST/core/datafiles/upload-spectra-csv Upload and parse spectra CSV file
  9. POST/core/datafiles/export-spectra-files
  10. GET/core/experiments/funding Search funding
  11. POST/core/germplasm/import Add or update many germplasms
  12. POST/core/germplasm/check check germplasms exist
  13. GET/core/locations/history Search location history of an object
  14. POST/core/locations/targetLocations Search the last geospatialized location of a target
  15. GET/core/locations/count Count locations
  16. GET/core/facilities/with_location Get only a list of facilities with a position (address/spatial coordinates
  17. GET/core/sites/with_location Get only the list of sites with a location
  18. GET/core/uri_search/{uri} Get a list of objects that match the passed URI
  19. GET/dataverse/RechercheDataGouvBasePath Get the Recherche Data Gouv url from the instance configuration
  20. POST/dataverse Create experiment as a Dataset
  21. GET/dataverse/datasetLanguages Get the available dataset languages
  22. GET/dataverse/datasetMetadataLanguages Get the available dataset metadata languages
  23. DELETE/core/datafiles/{uri} Delete a datafile
  24. POST/ontology/uris_labels Return associated rdfs:label of uris if they exist
  25. POST/ontology/uri_types Return all rdf types of some URIS

What's Deprecated⇑

  1. GET/ontology/uris_labels Return associated rdfs:label of uris if they exist
  2. GET/ontology/uri_types Return all rdf types of an uri
